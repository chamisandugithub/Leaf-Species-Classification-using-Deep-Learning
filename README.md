
##  Project Overview

This project combines **Random Forest (RF)** and **Convolutional Neural Networks (CNN)** to classify plant leaf species from both tabular features and leaf images.

The dashboard provides an interactive interface for exploring leaf data, visualizing clusters, and predicting species using multiple model approaches.

##  Features

-  Leaf species prediction using CNN image classification  
-  Random Forest classification using tabular leaf features  
-  Hybrid CNN + RF fusion model for improved prediction  
-  Interactive dashboard built with Streamlit  
-  Dimensionality reduction visualization (UMAP, PCA, t-SNE)  
-  Leaf thumbnail cluster visualization  
-  Image upload for real-time prediction  
-  Feature extraction using image masks

##  Real-World Impact

This system helps in plant species identification, agricultural research, biodiversity monitoring, and botanical education through AI-powered classification.
